软件安装

代码编辑器

代码编辑器推荐：可以下载个 【Visual Studio Code】，可以代码高亮，下载安装后想设置中文界面，打开软件点击左侧最下拓展程序，搜索 Chinese (Simplified) 进行安装，之后重启编辑器即可。撰写文章的时候想实时预览的话使用快捷键ctrl+k，然后按v键即可

Git

我们网站在本地的搭建与同步到GitHub上，都需要用到Git，从Git官网下载：【Git Download】

安装Node.js

Hexo基于Node.js，Node.js下载地址：【下载 | Node.js】 。

网站搭建

GitHub创建个人仓库

登录到GitHub,如果没有GitHub帐号，邮箱注册一个：【GitHub】 点击GitHub中的New repository创建新仓库，仓库名应该为： 你的用户名 .http://github.io。不可更改，比如我的仓库名为：**zuochao44.github.io**

创建本地文件夹

找一个地方放置你的博客文件，建议新建文件夹的名称为Blog，在文件夹内右键菜单内打开Git Bash

设置user.name和user.email配置信息：

生成ssh密钥文件：

之后根据提示按三次回车即可，默认不需要设置密码

然后找到生成的.ssh的文件夹中的id_rsa.pub密钥，将内容全部复制（具体位置请查看Git窗口，默认文件地址：C:\Users\用户名.ssh）

打开GitHub_Settings_keys 页面，新建new SSH Key

Title为标题，任意填即可，将刚刚复制的id_rsa.pub内的内容粘贴进去，最后点击Add SSH key。
最后在Git Bash中检测GitHub公钥设置是否成功，输入 ssh git@github.com ：

获得个人网站域名

域名是网站的入口，比如百度官网的域名是：baidu.com，常见的后缀有.com .top .cn .net .org等后缀，根据你自己的喜好，选择不同的后缀，最常见的就是.com后缀，当然这个也是比较贵的，有些域名后缀在大陆是没办法备案的，需要注意的是，如果是国外服务器，无需备案也能使用，根据需求决定

申请域名的地方有很多，这里推荐阿里云：阿里云域名注册 购买域名这也是我们整个搭建过程中惟一一个需要花钱的地方。如果你已经有了空闲域名就无需购买，直接使用即可。
